# jsh

Shell interativo `jsh` do ecossistema JeffNix/Alinix.

## Recursos

- Prompt dinâmico com suporte a temas.
- Neofetch integrado.
- Carregamento de arquivo `.jshrc` com configurações e exportações.
- Fonte `AlinixLogo` embutida para renderização perfeita do logotipo.

## Compilação

```bash
cargo build --release
```
